# UnderMyStamp
![Gameplay](./GamePlay/GamePlay2.gif)

트릭컬 리바이브 미니게임 '결제 서류가 맛있는 건에 대하여'라는 도장 찍기 게임을 구현 

목표: 7일 내에 구현

개발 기간: 26.07.15 ~ 26.07.21 (7일), 하루 3시간 기준 약 21시간 소모

색깔에 맞는 스탬프를 찍고 높은 점수를 획득하기 위한 게임입니다

## 다운로드 및 플레이 방법
[PC 다운로드](https://drive.google.com/file/d/1k1w8cPq6J_tAk5OMh9ErRbNK0AFS30YO/view?usp=sharing)

[Mobile apk 다운로드](https://drive.google.com/file/d/1tlUUZppMMuzK0VSF9dOlbKx-zYwkQIpd/view?usp=sharing)

- 플레이어가 들고있는 도장 색에 따라 오른쪽 승인 버튼을 누르면 점수를 얻습니다
- 왼쪽 '도장 교체' 버튼을 통해 도장 색을 변경할 수 있습니다
- 12시 방향 Fever게이지가 전부 차면 Fever 상태가 되며, 이 경우 오른쪽 승인 버튼을 누르기만 해도 정답 처리로 진행됩니다
- 12시 방향 타이머가 전부 소모되면 게임 오버가 됩니다.
